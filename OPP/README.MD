2. 构造函数 →

| 阶段         | 构造函数类型              | 难度   | 学习重点             |
| ---------- | ------------------- | ---- | ---------------- |
| 1️⃣ 基础入门   | 默认构造函数              | ⭐    | 最基本的构造方式         |
| 2️⃣ 参数构造函数 | 普通构造函数（带参数）         | ⭐⭐   | 初始化属性            |
| 3️⃣ 简化写法   | 语法糖构造函数             | ⭐⭐   | 一行写完属性赋值         |
| 4️⃣ 命名构造函数 | Named Constructor   | ⭐⭐⭐  | 创建多种“不同初始化方式”的对象 |
| 5️⃣ 初始化列表  | Initializer List    | ⭐⭐⭐  | 初始化只读属性或父类属性     |
| 6️⃣ 常量构造函数 | Const Constructor   | ⭐⭐⭐⭐ | 创建不可变对象（提高性能）    |
| 7️⃣ 工厂构造函数 | Factory Constructor | ⭐⭐⭐⭐ | 控制实例创建（例如单例、缓存）  |

```js
第 1 步：默认构造函数
   ↓
第 2 步：带参数构造函数
   ↓
第 3 步：语法糖构造函数（this）
   ↓
第 4 步：命名构造函数（fromJson、empty等）
   ↓
第 5 步：初始化列表（final、计算属性）
   ↓
第 6 步：常量构造函数（const）
   ↓
第 7 步：工厂构造函数（factory）
```
