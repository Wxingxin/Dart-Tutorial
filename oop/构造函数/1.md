---

## 1️⃣ 构造函数的核心作用

**构造函数（Constructor）** 是用来 **创建类的实例对象并初始化它的状态（属性）的特殊方法**。

它的主要职责：

1. **分配和创建对象**

   * 当你写 `var p = Person();` 时，构造函数负责在内存里创建 `Person` 实例。

2. **初始化对象的属性**

   * 你可以通过参数或者默认值来给对象的字段赋值：

     ```dart
     class Person {
       String name;
       int age;
       Person(this.name, this.age); // 用传入参数初始化
     }
     var p = Person('Tom', 20);
     ```

3. **在对象创建时执行特定逻辑**（可选）

   * 比如检查参数合法性、设置默认值、打印日志等：

     ```dart
     class Person {
       String name;
       int age;
       Person(this.name, this.age) {
         if (age < 0) age = 0;
       }
     }
     ```

---

## 2️⃣ 构造函数 vs 普通方法

| 区别   | 构造函数               | 普通方法     |
| ---- | ------------------ | -------- |
| 名称   | 必须和类名相同（或命名构造函数）   | 任意名称     |
| 返回值  | **没有返回值**（总是返回类实例） | 可以有任意返回值 |
| 调用时间 | **对象创建时自动调用**      | 必须显式调用   |
| 作用   | 初始化对象              | 执行某种操作   |

---

## 3️⃣ 总结一句话

> **构造函数就是对象的“初始化器”**，它可以接收参数，也可以不接收，用来设置对象的初始状态，而不仅仅是“处理参数”。
